# calender_application
インターン研修の最終課題で制作したカレンダーアプリです。  
DBはDriftを使用し、状態管理はRiverpodを使用しています。


## MakeFileコマンド
```dart
make format
```

コードをフォーマットし、不要なインポートをソートし、静的解析を実行します。

format


```dart
make build
```
依存関係を取得し、ビルドランナーをクリーンアップし、ビルドを実行します。

```dart
make clean
```
Flutterプロジェクトをクリーンアップし、依存関係を取得します。
